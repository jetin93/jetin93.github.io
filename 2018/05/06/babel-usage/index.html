Babel快速入门

发表于 2018-05-06

什么是Babel？总的来说，Babel是一个通用的多用途JavaScript编辑器。通过Babel你可以使用（并创建）下一代的JavaScript，以及JavaScript工具。

每年JavaScript新的标准/提案和新的特性层出不穷，但这些标准得到支持却需要时间，而Babel可以让你提前（甚至数年）使用它们。

Babel把用户编写的“超前”的代码向下编译成已经被普及的版本，即随处可用的版本。这一过程叫做“源码”到“源码”的编译，也被成为“转译”，即转换加编译。

例如，Babel能将以下ES2015新的箭头函数语法：

const square = n => n * n;

转译为：

var square = function(n) {
    return n * n;
}

Babel还支持语法扩展，比如React所用的JSX语法。同时还支持静态类型检查的流式语法（Flow Syntax）。

Babel将自身分成了数个核心模块，任何人都可以利用这些核心模块去创建自己的插件。至于如何去创建一个Babel插件，我将写在在下一篇文章中。

安装Babel

Babel-Cli

虽然可以在全局环境中安装Babel-Cli，但是根据单个项目，进行本地安装会更好一些。这么做的主要原因是：

1. 同一机器上的不同的项目可以依赖不同版本的 Babel, 这允许你一次更新一个项目。
2. 这意味着在你的工作环境中没有隐含的依赖项。它将使你的项目更方便移植、更易于安装。

我们可以通过以下命令本地安装 Babel CLI:

npm install --save-dev babel-cli

安装完成之后，你的package.json文件应该是这样的：

{
    "devDependencies": {
+       "babel-cli": "^6.0.0"
    }
}
